🔄 КАК ВИКОРИСТОВУВАТИ
================================================================================

ПЕРЕДАЧА ПЕРЕКЛАДІВ У КОМПОНЕНТИ:

// Layout
<Header locale={locale} dict={dict} />
<Hero dict={dict} />
<Services dict={dict} />
<Footer dict={dict} />

// Компонент
interface HeaderProps {
  locale: string
  dict: any
}

export function Header({ locale, dict }: HeaderProps) {
  return (
    <nav>
      <Link href={`/${locale}/services`}>{dict.nav.services}</Link>
    </nav>
  )
}

================================================================================
📝 ДОДАВАННЯ НОВИХ ПЕРЕКЛАДІВ
================================================================================

1. Додати ключ в uk.json:
   {
     "myFeature": {
       "title": "Мій заголовок",
       "description": "Опис"
     }
   }

2. Додати ключ в en.json:
   {
     "myFeature": {
       "title": "My Title",
       "description": "Description"
     }
   }

3. Використовувати в компоненті:
   <h1>{dict.myFeature.title}</h1>
   <p>{dict.myFeature.description}</p>
